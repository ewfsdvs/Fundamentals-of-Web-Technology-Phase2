# Premium Motors - Used Car Marketplace
## Project Introduction
This is a used car marketplace website built with pure HTML, CSS and JavaScript. It is a completion result of Web Technology Basic Course Phase A Delivery 2. The whole project is developed with generative AI tools, and all source codes are generated by artificial intelligence.

## Core Functions
### Main Features
Home page to display featured vehicles and search entry
User registration and login functions with local storage data simulation
Vehicle search system with filters for keywords, brands, price, production year and vehicle condition
Detailed vehicle information display and seller contact display
Vehicle posting function for registered and logged in users

### Technical Advantages
Pure front end development without external server support
Responsive layout for desktop and mobile devices
Standard form verification with regular expressions
Local storage for persistent data saving
Modular programming structure for JavaScript
